# eminem-sorter

## Overview
This web app presents the user one question at a time.
Each question follows the same format, two eminem songs are presented at a time and the user is expected to choose their favorite of the two.
After selection, the user is presented with another question of the same format recursively until enough questions have been answered to form an ordered list from the most favorite to least favorite of the song list.

## Technical
This web app runs with the help of Node, react, & webpack.
